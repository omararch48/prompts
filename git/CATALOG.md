# Catálogo: git/

## Propósito
Prompts para tareas relacionadas con el sistema de control de versiones Git: configuración, flujos de trabajo y generación de archivos de soporte.

## Inventario

| Nombre base | Estado | Descripción | Formatos disponibles |
|---|---|---|---|
| prompt_git_base | BORRADOR | Estructura base para prompts de tareas Git genéricas | txt, md, xml, json |
| prompt_gitconfig | BORRADOR | Prompt para asistir en la configuración de `.gitconfig` | md |
| prompt_gitignore | BORRADOR | Prompt para generar un archivo `.gitignore` personalizado | md |

## Notas de uso
- Todos los prompts de este directorio están en estado `BORRADOR` y no son funcionales aún.
- `prompt_git_base` puede usarse como plantilla estructural para crear nuevos prompts de Git.
